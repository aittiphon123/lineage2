# Event Template Addon

Use this template as a starting point for new activities/events.

## Quick start
1. Copy this folder:
   - `addons/event_template` -> `addons/<your_event_name>`
2. Edit `config/<YourEvent>.ini`
3. Add runtime scripts to `scripts/`
4. Add SQL migrations in `sql/` if needed
5. Implement `tools/deploy-addon.sh`
6. Register addon name in `addons/manifest/addons.manifest.ini`
