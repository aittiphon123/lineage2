# Catch-Up EXP/SP Usage Guide

## Overview
Catch-Up EXP boosts progression for low/mid-level players.

## Key Config (`config/Custom/CatchUpExp.ini`)
- `EnableCatchUpExp`
- `CatchUpLowMaxLevel`, `CatchUpMidMaxLevel`, `CatchUpMaxLevel`
- `CatchUpExpMultiplierLow/Mid`, `CatchUpSpMultiplierLow/Mid`
- `EnableRestedBonus`, `RestedMinOfflineHours`, `RestedExpMultiplier`, `RestedSpMultiplier`

## Tuning Strategy
- Keep low-level multipliers higher than mid/high.
- Use rested bonus modestly to avoid economy inflation.
- Re-check level brackets after server rate changes.
